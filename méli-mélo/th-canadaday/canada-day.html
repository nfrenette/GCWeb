Red background and white text - Canada Day 2020 theme

Use to apply the Canada Day 2020 theme of white text on a red background.

Appearance

This text is styled with .text-white and .bg-red.

CSS classes

.bg-red
Set a background color to red
.bg-red-carousel
Set a carousel background color to red
hr.brdr-red
Set border colour to red for hr
.panel-red
Set border colour to red for panel
.text-white
Set a font colour to white. (Not defined here; this is already included in WET.)

Correct use

Compliance point(s):

  • Use to introduce contrast to information
  • Use .text-white in combination with .bg-red to ensure sufficient contrast between text and background
  • Wrap text in a <span> tag if a style does not appear as it is meant to, due to specificity reasons

Incorrect use

Compliance point(s):

  • Do not use this component in a way that conflicts with the preceding compliance point(s)
  • Do not use as the only way to communicate information or intent, as colour alone is not accessible
